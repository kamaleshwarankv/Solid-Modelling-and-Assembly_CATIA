TITLE OF THE PRODUCT: IC ENGINE PISTON AND CRANKSHAFT (SOLID MODEL)

DESCRIPTION:
Designed a 3D model of an IC engine piston and crankshaft assembly using 3D experience by applying solid modeling techniques.
The crankshaft was created using extrusion and revolve features to achieve accurate geometry of crank webs, journals, and shaft.
The piston components were developed using solid features ensuring proper alignment with the crankshaft through connecting rods.
Assembly was done by positioning each component precisely to simulate the working mechanism of an internal combustion engine.
Focus was given to maintaining dimensional accuracy, smooth motion relationship, and realistic representation of engine components.

CHALLENGES FACED TO DEVELOP:
Maintaining correct alignment between piston, connecting rod, and crankshaft was difficult.
Creating complex crankshaft geometry with multiple offsets and rotations.
Ensuring proper dimensions and clearances between moving parts.
